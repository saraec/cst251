Lab 9: Styling Block Elements

Subject of the Lab

The subject of this lab is to use CSS to style my page as well as individual block elements.

Summary and Difficulties

I partnered with Jasmine for this lab. We had to work remotely and communicated over text. I recycled the code I have used in the preceeding labs. I changed almost all of the features in my CSS file. I increased the margins. I decreased the border size, padding, and letter spacing. I also wanted to try to apply an image to the background instead of floating in the corner. I modified the font colors and imported a font from Google Fonts for my headings. I used id=main for my main content and class=paragraph for my larger areas of text that I could apply CSS to the individually.
