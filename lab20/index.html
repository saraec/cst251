Lab 20: Team Project Sketches

Subject of the Lab

The subject of this lab is to create sketches of site layouts and receive feedback from group members.

Summary and Difficulties

Jasmine and I worked on the lab together after class on Wednesday. I linked the jquery, popper, and Bootstrap library in my CSS. Then I put the Bootstrap grid layout and nav bar into my html. Some of my content, like my headings and colors, were left over from the previous lab. I added a new font, new images, and new links. The only challenge I faced was trying to understand what and how elements defined in Bootstrap, so I could change the colors, size, etc. The only way I found to do this was to inspect elements in Chrome and copy and paste the CSS from there.



Results

This lab went a lot easier than the other labs because a majority of the CSS was provided by Bootstrap. I really liked using the grid layout because I struggled with creating a responsive layout in the previous labs, and Bootstrap did all the hard work for me.
